函数用法：

cm.worldSpouseMessage(type, message); //全服公告
cm.spouseMessage(type, message); //个人提示

下面是type的值所对应的喇叭类型：

* 0x00 = 白色
* 0x01 = 绿色
* 0x06 = 灰色
* 0x07 = 黄色
* 0x08 = 浅黄
* 0x09 = 蓝色
* 0x0A = 白底黑字
* 0x0B = 粉色
* 0x0C = 白底蓝字
* 0x0D = 粉底深紫色字
* 0x0F = 深黄底黑字
* 0x10 = 紫色
* 0x11 = 绿底黑字
* 0x14 = 浅蓝色
* 0x16 = 蓝黄蓝3次
* 0x19 = 浅紫色
* 0x1B = 浅黄色
* 0x1C = 深黄色黑体
* 0x1E = 深紫色
* 0x1F = 浅绿色
* 0x20 = 白底粉字
* 0x21 = 红底黑字
* 0x23 = 红底黄字
* 0x24 = 粉底黑字
* 0x25 = 浅黄底黑字
* 0x26 = 红底浅黄字
* 0x28 = 深红底黑字
* 0x2A = 深红色
